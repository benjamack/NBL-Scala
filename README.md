# NBL-Scala

Repositorio de contexto para el **New Business Launchpad (NBL)** — Trabajo Final de Graduación del MIA (Universidad de San Andrés), proyecto **Scala**. Equipo: Benjamín Mackinnon y Nico.

Este repo es **solo contexto y entregables académicos del NBL**. No es el código de la app. El desarrollo del producto vive en el repo separado **App-garmin** (no tocar ese repo para trabajo de tesis, ni este para features de producto — se comunican, pero son proyectos distintos).

## Estructura

- **`Contexto General/`** — `CLAUDE.md` (contexto NBL + resumen de la idea, fuente canónica del One Pager), `CLAUDE-scala-app-summary.md` (resumen técnico de la app para referencia), `design.md` (sistema de marca), `Guia/` (guía TFG MIA, roadmap, handbook de mentores).
- **`Modulo 1 - Ideacion/`** — One Pager y PPT del Módulo 1.
- **`Modulo 2 - Evaluacion de la oportunidad/`** — Doc #2, Doc #3 (evaluación de oportunidad) + anexo de investigación de mercado, modelo de unit economics y figuras.
- **`Modulo 3 - El cliente/`** — consignas del módulo, entregable (transcripts de entrevistas + resultados de encuesta), y el framework/plan de trabajo (guía de entrevistas, arquetipo de cliente, encuesta cuantitativa, tracker).
- **`Tesis (TFG)/`** — el escrito final de tesis. Todavía no existe un documento consolidado (a jul 2026 solo están los entregables por módulo) — se arma acá a medida que avance.

## Convención

Cada módulo nuevo del NBL suma su propia carpeta (`Modulo N - Nombre/`) siguiendo la numeración oficial del programa. El aprendizaje de cada módulo que deba arrastrarse hacia adelante (afinamientos de la idea, hallazgos de campo) se documenta en `Contexto General/CLAUDE.md`.
